#Project Overview
This project analyzes credit card customer financial data to understand spending behavior, revenue generation, credit utilization, and customer demographics.
The dashboard provides an interactive and executive-level view of customer activity, enabling data-driven decision-making for financial strategy and risk monitoring.
The solution is built in Power BI with recruiter-level dashboard design suitable for MNC analytics roles (aligned with the portfolio youâ€™re building).

# Business Objectives
The dashboard answers key analytical questions:
How is customer spending distributed across categories?
Which customer segments generate the highest revenue?
What are the transaction trends across time periods?
How do demographics influence credit card usage?

Where are potential credit risk indicators?
